## Install Autopsy for Linux

1.Open Terminal.

2.Update your package list: sudo apt update

3.Install Autopsy: sudo apt install autopsy

4.Launch Autopsy using: autopsy

5.Open http://localhost:9999/autopsy in a web browser.
